Vectors

Vectors are like dynamic arrays that can resize automatically at runtime.

  • Like arrays, they store elements in contiguous memory blocks.
  • Header file: #include <vector>
  • vector<int> vec; — Declare an empty vector of integers
  • vector<int> vec = {1, 2, 3, 4}; — Initialization with values
  • vector<int> vec(5, -1); — Vector of size 5, all elements initialized to -1
  • vec[0] — Accessing element at index 0
  • vec.size() — Get current number of elements

Vector Memory Implementation

  • Internally uses a dynamic array in the heap
  • When more space is needed, it doubles its capacity and copies elements to a new block
  • This causes occasional O(n) time when resizing, but average insertion time remains O(1)
        // Example
        vector vec;
        vec.push_back(10); // capacity = 1
        vec.push_back(20); // capacity doubled to 2
        vec.push_back(30); // capacity doubled to 4

Built-in Functions

  • vec.capacity() — Total allocated space (may be more than size)
  • vec.size() — Number of elements currently in the vector
  • vec.push_back(element) — Add element at the end
  • vec.pop_back() — Remove the last element
  • vec.clear() — Removes all elements

Note:

  • There is no push_front() or pop_front() in vector. Use deque or list for those.
