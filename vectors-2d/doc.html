2D Vectors

Concept

  • 2D vectors are vectors of vectors — each row is a separate vector.
  • They offer dynamic sizing and flexibility compared to arrays.
  • vector<vector<int>> matrix; — declaration of a 2D vector.
  • matrix[i][j] — access a specific element.
  • matrix.size() — gives the number of rows.
  • matrix[i].size() — gives number of columns in the i-th row.

Example

        vector<vector<int>> matrix = {{1, 2, 3}, {4, 5, 6}};

        for(int i = 0; i < matrix.size(); i++) {
            for(int j = 0; j < matrix[j].size(); j++) {
                cout << matrix[i][j] << " ";
            }
            cout << endl;
        }

Note:

  • Unlike 2D arrays, each row in a 2D vector can have a different number of elements.

Example of irregular 2D vector:

        vector<vector<int>> matrix = {
            {1, 2, 4, 5},
            {6, 7},
            {3, 8, 9}
        };
