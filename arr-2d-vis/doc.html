🧱 2D Arrays

2D arrays are like arrays inside arrays — they have rows and columns.

📌 Initialization

int arr[2][3] = {
        {1, 2, 3},
        {4, 5, 6}
    };
  • Think of it as a grid or table.
  • arr[1][0] accesses a single element.
  • Data is stored in a linear memory block.

✍️ Input & Output

➡️ Input:


    for(int row = 0; row < rowSize; row++) {
        for(int col = 0; col < colSize; col++) {
            cin >> arr[row][col];
        }
    }

➡️ Output:


    for(int row = 0; row < rowSize; row++) {
        for(int col = 0; col < colSize; col++) {
            cout << arr[row][col] << " ";
        }
        cout << endl;
    }

💾 Memory Allocation

  • 2D arrays use a continuous block of memory.
  • Row-Major Order (default in C++) — elements stored row by row.
  • Column-Major Order — used in some other languages, stored column by column.
