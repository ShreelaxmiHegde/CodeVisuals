2D Dynamic Arrays

Concept

  • 2D dynamic arrays are created by combining multiple 1D dynamic arrays.
  • They cannot be created directly like static 2D arrays.
  • Each row is a pointer to a dynamically allocated 1D array.
  • int** arr = new int*[rows]; â€” creates an array of row pointers.
  • Each arr[i] points to a 1D array of columns.
  • Memory is allocated in the heap.
  • Memory is not contiguous like static 2D arrays.

Memory Allocation Process

  • Step-by-step memory creation:
        int row, col;
        cin >> row >> col;

        // Step 1: Allocate memory for row pointers
        int** matrix = new int*[row];

        // Step 2: Allocate memory for each row
        for(int i = 0; i < row; i++) {
            matrix[i] = new int[col];
        }

        // Step 3: Storing values
        int x = 1;
        for(int i = 0; i < row; i++) {
            for(int j = 0; j < col; j++) {
                matrix[i][j] = x++;
            }
        }

Key Points

  • You can access elements like matrix[i][j].
  • Always free the memory after use to avoid memory leaks.
        // Freeing memory
        for(int i = 0; i < row; i++) {
            delete[] matrix[i]; // Delete each row
        }
        delete[] matrix; // Delete the row pointers
