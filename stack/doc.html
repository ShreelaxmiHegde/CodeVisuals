Stacks

Definition

  • LIFO (Last In First Out) data structure.
  • stack<int> s; initialization using STL.
  • Elements are added and removed only from the top of the stack.
  • s.push() adds an element at the top.
  • s.pop() removes the top element.
  • s.top() returns the top element.
  • All operations have constant time complexity O(1).

Implementation

*A stack can be implemented using arrays, vectors, linked lists, etc.

1. Using STL

        #include <stack>

        stack<int> s;

2. Using Vector

        class Stack {
        private:
            vector<int> vec;

        public:
            void push(int val) {
                vec.push_back(val);
            }

            void pop() {
                if (isEmpty()) return;
                vec.pop_back();
            }

            int top() {
                if (isEmpty()) return -1;
                return vec[vec.size() - 1];
            }

            bool isEmpty() {
                return vec.size() == 0;
            }
        };

3. Using Vector with class template

*Works for all data types

        template<class T>
        class Stack {
        private:
            vector<T> vec;

        public:
            void push(T val) {
                vec.push_back(val);
            }

            void pop() {
                if (isEmpty()) return;
                vec.pop_back();
            }

            T top() {
                if (isEmpty()) return T();
                return vec[vec.size() - 1];
            }

            bool isEmpty() {
                return vec.size() == 0;
            }
        };

4. Using Linked List

        template<class T>
        class Stack {
        private:
            List<T> ll;

        public:
            void push(T val) {
                ll.push_front(val);
            }

            void pop() {
                ll.pop_front();
            }

            T top() {
                return ll.front();
            }

            bool isEmpty() {
                return ll.size() == 0;
            }
        };
